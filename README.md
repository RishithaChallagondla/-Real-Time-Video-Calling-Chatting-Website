# Google Meet Clone

A real-time video conferencing web application inspired by Google Meet. This project enables users to create and join video meetings with audio, chat, and screen sharing capabilities.

---

## Features

- **Real-time Video & Audio:** High-quality video and audio conferencing.  
- **User Authentication:** Secure login and registration system.  
- **Meeting Scheduling:** Create and join meetings easily.  
- **Screen Sharing:** Share your screen during meetings.  
- **Chat Functionality:** Real-time messaging during meetings.  

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (JSON Web Tokens)  
- **Real-time Communication:** WebRTC
